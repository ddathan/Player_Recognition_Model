run_team_and_player_model.py

Python script to run the saved team and player recognition model on a folder images

Call from command line with path to directory of images
eg. run_team_and_player_model.py part1

Directory of images must be in format team\player\image.jpg

Requires team_and_person_model_python.py model and team_and_person_model_labels.json to be in the same folder as it is run in

Outputs the number of correct identifications and percentage correct
